# 2-3 阶梯训练：半音阶爬行

欢迎来到第一个真正的技术关卡。这里没有华丽的旋律，只有纯粹的手指力量与协调训练——这就是著名的“爬格子”。

## 科学视角：神经募集与肌肉记忆

爬格子练习的本质是**神经募集 (Neural Recruitment)**。通过高重复、慢速度的动作，你的大脑会建立从大脑皮层到指尖肌肉的专属信号通路。

*   **同步性**：左手按弦的瞬间与右手拨弦的瞬间必须完全重合。
*   **颗粒感**：每一个音符都应该是清晰、饱满且独立的。

## 关卡任务：1-2-3-4 练习

我们将使用 1 弦（最细的弦）进行基础练习：

1.  **食指**按 1 品，弹奏。
2.  **中指**按 2 品，弹奏。
3.  **无名指**按 3 品，弹奏。
4.  **小指**按 4 品，弹奏。

### 技术细节：
*   **按弦位置**：尽量靠近品丝（靠近琴箱方向），但不要压在品丝上。这样可以用最小的力量获得最清晰的声音。
*   **手指保留**：当你按 2 品时，食指不要抬起；按 3 品时，前两指保持在指板上。这能锻炼手指的独立性和张力。
