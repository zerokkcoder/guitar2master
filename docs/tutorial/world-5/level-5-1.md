# 5-1 绝望之墙：F和弦挑战

欢迎来到吉他手的“成人礼”。

## 科学视角：压强与杠杆

大横按 (Barre Chord) 需要食指同时压住六根琴弦。根据物理学公式 **P = F / S**，为了获得足够的压强，我们需要：
1.  **减小接触面积 (S)**：使用食指的侧面（较硬的部分）而不是指腹。
2.  **利用杠杆原理**：大拇指作为支点，放在琴颈背部的中心位置。

## 关卡任务：攻克技巧

1.  **侧压法**：食指微向左侧倾斜，利用骨头较硬的地方按弦。
2.  **手肘辅助**：右手肘轻微向后拉琴箱，利用杠杆力帮助左手按弦，而不是单纯靠手指捏。
3.  **高度降低**：如果按不动，检查琴颈是否太高，或者弦距是否过大。
