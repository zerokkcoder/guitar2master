# 3-3 循环迷宫：常用和弦走向

为什么有些歌曲听起来感觉很像？因为它们使用了相似的“和弦走向”。

## 科学视角：调性引力

在音乐理论中，不同的和弦在调式中扮演着不同的角色：
*   **Ⅰ 级 (主和弦)**：家，最稳定的感觉。
*   **Ⅳ 级 (下属和弦)**：离家，稍微有些张力。
*   **Ⅴ 级 (属和弦)**：极大的张力，迫切想要回到家。

## 万能走向：1-6-4-5

在 C 大调中，这个走向是：
**C -> Am -> F (或 Fmaj7) -> G**

```text
       C (1)     Am (6)    Fmaj7 (4)    G (5)
    |----0----|----0----|----0----|----3----|
    |----1----|----1----|----1----|----0----|
    |----0----|----2----|----2----|----0----|
    |----2----|----2----|----3----|----0----|
    |----3----|----0----|----x----|----2----|
    |----x----|----x----|----x----|----3----|
```

## 关卡任务：循环弹奏

1.  每个和弦弹奏 4 拍。
2.  尝试循环这个走向 4 次，不要中断。
3.  闭上眼睛，感受每个和弦带来的情绪变化。
