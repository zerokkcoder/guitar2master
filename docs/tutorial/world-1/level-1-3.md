# 1-3 均衡之塔：持琴秘籍

错误的姿势会导致肌肉疲劳甚至受伤。这一关我们将从人体工程学的角度，构建稳固的“演奏塔”。

## 科学视角：重心与力矩

持琴的核心在于找到吉他与身体的**三个接触点**，形成稳定的三角形，从而释放双手的负担：

![持琴姿势示意图](../../images/guitar-posture.svg)

1.  **接触点 A**：琴箱凹陷处放在右腿上（或左腿，如果你使用古典姿势）。
2.  **接触点 B**：右前臂搭在琴箱最凸出的边缘。
3.  **接触点 C**：胸部轻微接触琴箱背面。

## 关卡要点：三不原则

*   **不低头**：不要过度弯腰看指板，这会导致颈椎受压。尝试微调吉他倾斜角度，用余光观察。
*   **不耸肩**：双肩保持自然下垂，消除紧张感。
*   **不夹臂**：右手肘部应自然外展，不要紧贴身体，给拨弦留下空间。

## 互动练习：坐姿检查表
*   [ ] 椅子高度适中（大腿与地面平行）。
*   [ ] 背部挺直，不要靠在椅背上。
*   [ ] 琴头稍微向上倾斜（约 30-45 度）。
