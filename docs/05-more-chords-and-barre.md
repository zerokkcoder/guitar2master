# 第五章：进阶和弦与大横按

如果你能坚持到这里，恭喜你，你将迎来吉他学习路上的第一个“大BOSS”——大横按。

## 1. 更多常用和弦

### Dm 和弦
*   食指：1 弦 1 品
*   中指：3 弦 2 品
*   无名指：2 弦 3 品

### E7 和弦 (常用于转调或蓝调)
*   食指：3 弦 1 品
*   中指：5 弦 2 品
*   (4 弦、2 弦空弦)

## 2. 攻克 F 和弦 (大横按)

F 和弦是 C 大调的 IV 级和弦，非常重要。

### 指法
*   **食指**：横按 1 品所有六根弦 (Barre)。
*   **中指**：3 弦 2 品。
*   **无名指**：5 弦 3 品。
*   **小指**：4 弦 3 品。

### 为什么这么难？
因为食指需要同时按住多根弦，对指力要求较高。

### 攻略技巧
1.  **侧面按弦**：食指稍微侧一点，用骨头比较硬的一侧去接触弦，而不是肉肉的指肚。
2.  **靠近品丝**：食指尽量靠近 1 品品丝。
3.  **拇指位置**：左手拇指应顶在琴颈背面的中下部，与中指对应，提供反向支撑力。
4.  **简化版 (小 F)**：
    *   食指：同时按 1 弦 1 品和 2 弦 1 品。
    *   中指：3 弦 2 品。
    *   无名指：4 弦 3 品。
    *   (不弹 5、6 弦)。

## 3. 经典和弦走向练习

**卡农走向 (Canon in D)**：
C - G - Am - Em - F - C - F - G

试着配合扫弦弹奏这个走向，你会发现你可以唱出几百首流行歌（如《小情歌》、《各种万能和弦串烧》）。
